Generación de datos:
rng(1); % Control de aleatorios
r = sqrt(rand(100,1)); % Radio
t = 2*pi*rand(100,1); % Anglulo
data1 = [r.*cos(t), r.*sin(t)]; % Puntos
r2 = sqrt(3*rand(100,1)+1);
t2 = 2*pi*rand(100,1);
data2 = [r2.*cos(t2), r2.*sin(t2)];
Ahora, veamos gráficamente como están distribuidos los datos:
figure;
plot(data1(:,1),data1(:,2),'r.','MarkerSize',15)
hold on
plot(data2(:,1),data2(:,2),'b.','MarkerSize',15)
axis equal
hold off
Ahora, pondremos la información de ambas clases y un clasificador.
data3 = [data1;data2];
theclass = ones(200,1); %Un vector con la clase binaria 1 y -1. Clasificador.
theclass(1:100) = -1;
Ahora entrenaremos el modelo SVM con la función kernel de tipo RBF, y box contraint, y mostraremos sus resultados.
% Entrenar el modelo SVM
cl = fitcsvm(data3,theclass,'KernelFunction','rbf',...
'BoxConstraint',Inf,'ClassNames',[-1,1]);
% Predecir puntajes en la gráfica
d = 0.02;
[x1Grid,x2Grid] = meshgrid(min(data3(:,1)):d:max(data3(:,1)),...
min(data3(:,2)):d:max(data3(:,2)));
xGrid = [x1Grid(:),x2Grid(:)];
[~,scores] = predict(cl,xGrid);
% Graficar los datos junto con el limite clasificador.
figure;
h(1:2) = gscatter(data3(:,1),data3(:,2),theclass,'rb','.');
hold on
h(3) = plot(data3(cl.IsSupportVector,1),data3(cl.IsSupportVector,2),'ko');
contour(x1Grid,x2Grid,reshape(scores(:,2),size(x1Grid)),[0 0],'k');
legend(h,{'-1','+1','Support Vectors'});
axis equal
hold off
Es turno ahora de entrenar el modelo SVM con la función kernel, con parámetros por defecto.
cl2 = fitcsvm(data3,theclass,'KernelFunction','rbf');
[~,scores2] = predict(cl2,xGrid);
